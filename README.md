# Warrant Python Library

Use [Warrant](https://warrant.dev/) in Python projects.

## Installation

```python
pip install warrant-python
```

## Usage

```python
import warrant

warrant = Warrant("api_test_f5dsKVeYnVSLHGje44zAygqgqXiLJBICbFzCiAg1E=")
warrant.create_user()
```
